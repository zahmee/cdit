# سياق: حملة Google Ads + تتبّع التحويلات لموقع cdit.co

> هذا الملف يلخّص قرارات اتُّخذت في جلسة سابقة، لإكمال العمل في Claude Code.

## المنتج
- نظام محاسبي / ERP سعودي، **معتمد ومتوافق مع ZATCA** (الفاتورة الإلكترونية).
- باقتان:
  - **المخفّف**: 300 ريال/سنة — للمنشآت الصغيرة والمتوسطة (مقاولات، محاماة، مكاتب هندسية، دعاية وإعلان). **الأكثر طلباً.**
  - **الكامل**: 900 ريال/سنة — لمن يحتاج مخازن/مستودعات (صيدليات وغيرها).
- السوق: السعودية. اللغة: العربية.
- الموقع: cdit.co (Angular + FastAPI، نشر Docker على VPS).

## الهدف
حملة Google Ads مربحة. **القيد الحاكم:** قيمة العميل منخفضة (300/900 ريال/سنة)،
لذا لا نبيع من النقرة مباشرة — لازم **التقاط بيانات العميل** ومتابعته.

## القرارات المتّفق عليها
1. **تجنّب الكلمات العامة الغالية** (`برنامج محاسبة`, `ERP`). النقرة غالية والعائد سالب عند هذا السعر.
2. **التركيز على كلمات ZATCA/فاتورة** + الكلمات الطويلة المحددة بكل قطاع.
3. **هدف التحويل = طلب تجربة مع التقاط بيانات** (اسم + جوال + نوع النشاط)، وليس مجرد دخول ديمو.
4. الديمو المشترك = زر ثانوي فقط (لا يلتقط بيانات، لا يصلح كتحويل).
5. نبدأ بحملة واحدة (ZATCA + المخفّف)، Search فقط، 30–50 ريال/يوم، Maximize Conversions، ونتركها تتعلّم 2–3 أسابيع.

## نقطة الالتقاط المقترحة (عبر "مساعد إبداع الذكي" الموجود على الموقع)
```
إعلان → صفحة الموقع → المساعد يسأل عن النشاط
       → المساعد يطلب الاسم + الجوال  ←── نقطة التحويل
       → عند إدخال الجوال:
            ✅ تسجيل حدث تحويل في Google Ads
            ✅ حفظ البيانات في قاعدة البيانات (لمتابعة العميل)
            ✅ فتح التجريبي/الديمو المناسب للقطاع
```

## المطلوب إنجازه في Claude Code
1. فحص بنية المساعد الذكي على الموقع (Angular component أم سكربت خارجي؟).
2. إضافة خطوة التقاط (اسم + جوال + نوع النشاط) داخل المساعد.
3. إنشاء endpoint في FastAPI لحفظ الـ Lead + إشعار صاحب الموقع.
4. ربط **حدث تحويل Google Ads** يُطلق عند نجاح إدخال الجوال (gtag conversion event).
5. تركيب **Google Tag** على الموقع والتحقق منه عبر Tag Assistant.

## ملاحظة استراتيجية
عند سعر 300 ريال، يُنصح بدمج الإعلانات مع **SEO على كلمات ZATCA** (مجاني ومستدام)،
واعتبار Google Ads ماكينة تجارب سريعة لإثبات الطلب. النظام الكامل (900 ريال) يستحق ميزانية أعلى.
